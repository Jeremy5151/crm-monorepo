# CRM Monorepo

## 🚀 Quick Start

```bash
# Install dependencies
pnpm install

# Start all services
pnpm dev:all
```

## 📁 Project Structure

- `apps/web` - Next.js frontend
- `apps/api` - NestJS backend
- `packages/` - Shared packages

## 🔧 Development

- Frontend: http://localhost:3000
- API: http://localhost:3001
- Database: PostgreSQL (Docker)

## 📋 Features

- Lead management
- Broker integrations
- Template system
- Bulk operations
- Real-time status updates
